Term - samling link

Properties
Name Value
Description

Søk etter kilder (flere søkekriteria som er satt sammen). Det er ikke det aktuelle søkeresultatet, men en rekke av sammensatte søkebetingelser som kan kjøres ved behov.

Brukes ved opprettelse av nye arbeidslister, for å spare tid ved inntasting av nye søk.

Eksempel:

* Alle kilder som mangler årstall

* Alle kilder som har emneknagger "Pantebok" og "Folketelling"
